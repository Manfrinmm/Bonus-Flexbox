fazer o documento ocupar 100% da tela(altura):
html,body,#app{ height:100%; margin:0; }

Display: flex

PROPRIEDADES:
flex-direction: row -> vão sempre seguir na mesma linha (padrão)
	''	row-reverse -> vão sempre seguir na mesma linha, porem inverso
	''	column -> vão sempre seguir na mesma coluna
	''	column-reverse -> vão sempre seguir na mesma coluna, porem inverso

flex-grow: 1 -> aceita ser aumentado para caber no container dele (ocupa toda largura disponivel em tela)
flex-shrink: 1 -> aceita ser espremido para caber no container (padrão)
flex: 1 0-> soma de grow e shrink, nessa ordem. 
flex: 1 -> permite aumentar e espremer

flex-wrap: wrap -> permite quebrar linha, primeiro elemento começa no inicio
flex-wrap: wrap-reverse -> permite quebrar linha, primeiro elemento começa no fim
ALINHAMENTO:
	row{
align-items -> alinhar verticalmente
justify-content -> alinhar horizontalmente
}
	column{
align-items -> alinhar horizontalmente 
justify-content -> alinhar verticalmente
}

PROPRIEDADES:

align-items: flex-start -> vão sempre se alinhar no começo (padrão)
justify-content: flex-start -> vão sempre se alinhar no começo (padrão)

align-items: center -> alinha conteúdo ao centro do espaço disponivel 
align-items: flex-end -> alinha conteúdo ao final do espaço disponivel

justify-content: center -> alinha conteúdo ao centro do espaço disponivel 
justify-content: flex-end -> alinha conteúdo ao final do espaço disponivel

justify-content: space-between -> coloca um espaço igual entre os itens
justify-content: space-araound -> coloca um espaço igual entre os itens com espaço no começo e fim


align-content -> alinhar elementos que estão em mais de uma linha
align-content: space-araound -> coloca um espaço igual entre os itens com espaço no começo e fim (padrão)
